# Docs

Astro Starlight documentation app for `svelte-jsonschema-form`.

## Content

- Documentation pages live in `src/content/docs`.
- Interactive demos live in `src/demos` and are referenced from MDX through the `Demo` component.
- The sidebar and Starlight configuration live in `astro.config.mjs`.
- Package changelog pages are loaded from package changelogs at build time.
