The `Labels` interface defines the text strings used throughout the form.

<Code code={code} lang="typescript" />

To use a translation, import a locale file and pass it to the form:

```ts
import { translation } from "@sjsf/form/translations/en";

const form = createForm({
  translation,
  // ... other options
});
```
