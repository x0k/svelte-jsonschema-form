You may want to use Zod validator because:

- You can use Zod schema as a source of truth for the form value type (`z.infer`).
- This is an easy way to add custom error messages

## Installation

<Npm pkg="@sjsf/zod4-validator zod" />

## Example

:::tip

Use `@sjsf/zod4-validator/mini` to work with `zod/v4-mini`

:::

## Async validation

This validator supports async validation.

```typescript
import { setupAsyncFormValidator } from "@sjsf/zod-validator/classic";

const { schema, validator } = setupAsyncFormValidator(zodSchema);
```
