  - `createPropertyId`
- Removed the following types:
  - `IdPrefixOption`
  - `IdSeparatorOption`
  - `IdPseudoSeparator`
  - `IdOptions`
  - `PathToIdOptions`
- `focus-on-first-error` module changes:
  - Updated signatures of `getErrorsList` and `getFocusAction`
  - Removed the `GetFocusActionOptions` type
